# MAX 26.22.0 V1 — Mods by JohNick

Релиз поверх стока **26.22.0 (build 6770)**. Приватность, нейтрализация телеметрии, UX-функции. **27 тумблеров**, все по умолчанию OFF (антидетект).

## 🆕 Новое в этой версии

- **🕶 Умная античиталка** — отметка «прочитано» уходит собеседнику **только после того, как вы ответили** в чат. До ответа читаете молча. Счётчик непрочитанных корректно сбрасывается и в списке чатов, и внутри чата.
- **Невидимка-каскад** — «Невидимка» одним переключателем взводит умную античиталку, антипечатанье и антиприсутствие. Дочерние флаги можно менять отдельно.
- **📎 Отправка любых файлов** (всегда включено) — `.apk`, `.exe`, `.jar` и др. Клиентский блок снят, серверный фильтр по расширению обходится авто-суффиксом **`.bin`** (25 типов). Получателю нужно убрать `.bin` из имени.
- **🔕 Три новых тумблера скрытия:** «Скрыть Max для бизнеса» (сервисный чат), «Скрыть Пригласить/Найти по номеру» (2 нижние ссылки в списке), «Только чаты» (спрятать каналы).
- **📲 Клон-push фикс** — уведомления на клон (`ru.oneme.ap2`) больше не требуют `pm clear`: device_id клона изолирован.
- **❗ Красный «!» фикс** — исчез ложный статус-ошибки исходящих в режиме Невидимка.
- **✅ Откат «Отметить непрочитанным»** в Невидимке устранён (read-ledger).

## ⚠ Честно

- Относительно 26.21.1 V2 **временно не активны** 4 транспортные нейтрализации (force TLS-validate, DoH off, WS-redirect guard, WS-опкод 5) — их якоря в стоке 26.22.0 переименованы R8, вернём в следующем обновлении. Базовые дефолты стока в этих точках безопасны.
- Полный `pm clear` клона теряет device_id для push — потребуется повторный вход по SMS.

## Установка

`install -r` поверх любой версии мода 26.x — данные сохраняются (та же debug-подпись). Первая установка: разрешить «неизвестные источники» → открыть APK → войти по номеру.

| APK | Кому |
|---|---|
| `MAX_26.22.0_arm64_V1.apk` | большинство телефонов (с 2018 г.) |
| `MAX_26.22.0_arm7_V1.apk` | старые 32-битные |
| `MAX_26.22.0_arm64_V1_clone.apk` | клон (2-й аккаунт), пакет `ru.oneme.ap2` |
| `MAX_26.22.0_arm7_V1_clone.apk` | клон для 32-битных |

Полный технический разбор — [`AUDIT_26.22.0_V1.pdf`](AUDIT_26.22.0_V1.pdf).
