# MAX 26.23.0 GP V4 — Mods by JohNick

**Дата**: 2026-07-17 · **Сток**: 26.23.0 (build 6777) · **Мин. Android**: 8.0 (API 26)

Хотфикс поверх V3. Три точечных фикса: обновлялка в клоне, ложное «доступно обновление», Watch-mode на компактных часах. Ставится поверх V3 через `install -r` без потери данных.

## 🐞 Фиксы V4

- **📥 Клон качал основной APK.** UpdateChecker не различал main / clone-пакет — при обновлении из клона (`ru.oneme.ap2`) скачивался обычный main-APK, и системный установщик предлагал поставить не тот пакет. Теперь picker asset'ов сверяет `getPackageName().endsWith("ap2")` с наличием `_clone` в имени файла: клон качает только `*_clone.apk`, main — только не-clone.
- **🔎 Ложное «доступно обновление».** `MIRROR_VERSION_TAG` в build-config не был бампнут с V2 (`26.23.0.2`), поэтому свежая V3-сборка сравнивала себя с mirror-тегом `26.23.0.3` и всегда показывала «есть обновление». Теперь `26.23.0.4` — синхронно с mirror-тегом релиза.
- **⌚ Watch-mode: SMS-код-инпут на OnePlus Watch 3 (466 px) не влезал** — density 240 dpi давал 310 dp высоты, поле ввода кода оказывалось за пределами экрана без возможности проскроллить. Плотность снижена до **200 dpi**, `smallestScreenWidthDp` = **373**. На OnePlus Watch 3 / Huawei Watch 3-4 Pro (466 × 466) доступно ≈373 × 373 dp — SMS-экран умещается. На телефонах (1080 px+) режим по-прежнему не активируется.

## ❓ Если видишь красный «!» на сообщении или чат открывается не внизу

Проверь **тумблер «Невидимка»** (`invisible`) — это мастер-переключатель, каскадно взводит `smartAntiRead`, `antiTyping`, `antiPresence`. При `invisible=ON`:

- Отметка «прочитано» уходит собеседнику **только после твоего ответа** (это `smartAntiRead`). Собеседник видит непрочитанным ⇒ у тебя чат может открываться в предпоследней позиции.
- Возможны отдельные ложные `!` на исходящих сообщениях (гейт `gateSendError` подавляет их, если реально сообщение доставлено).

Хочешь стандартное поведение — выключи `invisible` (и убедись что `smartAntiRead` тоже OFF).

## Установка

Ставится поверх V3 через `install -r` — данные сохраняются. Клон рядом с оригиналом.

- `MAX_26.23.0_arm64_V4.apk` — современные телефоны (2018+). **Если не уверены — этот.**
- `MAX_26.23.0_arm7_V4.apk` — старые 32-битные устройства.
- `MAX_26.23.0_arm64_V4_clone.apk` / `MAX_26.23.0_arm7_V4_clone.apk` — клон для второго аккаунта.
