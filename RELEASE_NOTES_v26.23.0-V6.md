# MAX 26.23.0 GP V6 — Mods by JohNick

**Дата**: 2026-07-20 · **Сток**: 26.23.0 (build 6777) · **Мин. Android**: 8.0 (API 26)

Хотфикс поверх V4. Один точечный фикс: часы (⏱) больше не висят на исходящих сообщениях при активной «Антиудалялке». Ставится через `install -r` без потери данных.

## 🐞 Фикс V6

- **⏱ → ✓ Часы на исходящих при antiDelete=ON.** Когда включена «Антиудалялка», на своих отправленных сообщениях висела иконка часов (pending delivery) — не проходила в галочку доставки. После рестарта MAX часы исчезали (реальный статус подтягивался из БД).
  Причина: WS-пакеты delivery-ack дропались нашим antiDelete-stub'ом вместе с delete-events, и in-memory состояние delivery оставалось `SENDING`. Теперь в маппере `Le0a → Lpz9` при `antiDelete=ON` состояние `SENDING` переклеивается в `SENT` (только оно; `ERROR/READ` не трогаем). БД не изменяется — visual-only.

## Установка

Ставится поверх V4 через `install -r` — данные сохраняются. Клон рядом с оригиналом.

- `MAX_26.23.0_arm64_V6.apk` — современные телефоны (2018+). **Если не уверены — этот.**
- `MAX_26.23.0_arm7_V6.apk` — старые 32-битные устройства.
- `MAX_26.23.0_arm64_V6_clone.apk` / `MAX_26.23.0_arm7_V6_clone.apk` — клон для второго аккаунта.
