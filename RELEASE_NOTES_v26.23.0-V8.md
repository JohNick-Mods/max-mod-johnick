# MAX 26.23.0 GP V8 — Mods by JohNick

**Дата**: 2026-07-23 · **Сток**: 26.23.0 (build 6777) · **Мин. Android**: 8.0 (API 26)

Крупное обновление поверх V7.1: выбор цвета иконки на рабочем столе (6 цветов, меняется на лету из настроек), исправлен порядок отправленных сообщений при включённой «Невидимке», устранены два краша (при запуске и при переключении на вкладку чатов), и «Невидимка» теперь работает единым тумблером. Ставится через `install -r` поверх V7.1 — данные сохраняются.

## 🆕 Изменения V8

- **🎨 «Цвет иконки» — 6 цветов на выбор.** В настройках мода — раздел «Цвет иконки»: синий, красный, зелёный, фиолетовый, чёрный, оранжевый. Меняется на лету, без переустановки — на рабочем столе иконка перерисовывается в выбранный цвет. Один APK, один пакет — никаких «сборок под цвет».
- **📤 Порядок отправленных при «Невидимке» — исправлен.** Раньше при включённой «Невидимке» три сообщения подряд часто приходили задом наперёд (сервер переприсваивал им время после отправки, а список в чате не пересортировывался). Теперь на входе адаптера список пересортировывается по времени и локальному id — тем же правилом, что применяет сток при выключенной «Невидимке». При выключенной «Невидимке» ничего не меняется (сток байт-в-байт).
- **🕹 «Невидимка» — единый тумблер.** Мастер-переключатель `invisible` теперь каскадно взводит и снимает всех детей (`antiRead` / `antiTyping` / `antiPresence`) — не нужно ходить по каждому тумблеру отдельно. Обратная совместимость сохранена: если раньше был включён только один из дочерних — он остаётся включённым.
- **🛡 Краш при запуске — устранён.** В сборках с включённым `startOnNew` (старт на папке «Новые») в редких случаях приложение падало на первом старте после установки. Проблемный вызов взят в защищённый блок — при сбое мод молча уступает штатному поведению вместо краша.
- **🛡 Краш при открытии вкладки «Чаты» — устранён.** Приложение падало с `Resources$NotFoundException` на некоторых устройствах при переключении на вкладку чатов. Причина — расхождение путей ресурсов после нормализации config-квалификаторов при пересборке; починено таблицей соответствия (28 путей), ресурсы теперь находятся штатно.

- `MAX_26.23.0_arm64_V8_clone.apk` / `MAX_26.23.0_arm7_V8_clone.apk` — клон для второго аккаунта.

## Известные ограничения 26.23.0 V8

- **Порядок отправленных при «Невидимке»** — исправлен в подавляющем большинстве случаев; в редких сценариях (задержка reconcile больше одного тика адаптера) отдельная пара сообщений может встать не по времени. Переоткрытие чата всегда возвращает правильный порядок.
- **Счётчик непрочитанных** на иконке чата может накрутиться на удалённое сообщение (сервер шлёт итоговое число без ID — точечно вычесть нельзя без отдельной доработки). Само сообщение видно в чате. *(запланировано в следующей версии)*
- **AMOLED** — покрытие ~95%: список чатов и всплывающие шторки могут остаться тёмно-серыми.
