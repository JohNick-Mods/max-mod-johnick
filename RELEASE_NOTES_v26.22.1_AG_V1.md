# MAX 26.22.1 — AppGallery V1 (2026-07-11)

**Первый релиз AG-редакции мода**, портированный с рабочей GP V3 26.22.1.

**Совместимость:** Android 8.0–16.0 · arm64-v8a + armeabi-v7a · основной + клон · **работает без Google Play Services** (Huawei HarmonyOS / Honor MagicOS / любые ROM без GMS).

## 🆕 Что нового в AG

- **Push через Huawei Push Kit (HMS)**, не через FCM — уведомления доходят на «безгуглых» Huawei.
- **Fat AppGallery APK → thin по ABI**: 148 MB → два universal APK по ~40 MB (arm64 + arm7).
- **4 DEX** (в GP было 3 — в AG отдельный `classes4.dex` для новых классов Huawei-стека).
- **Все 43 патча GP V3 портированы под новую R8-раскладку AG**.

## ✔ Работает как в GP-моде

- 🛡️ Анти-удаление переживает hard-kill (HMS-push `MessageRemoved` попадает в персистент).
- 💬 «Умная античиталка» — открытие чата сразу на последнем сообщении (не на первом непрочитанном). Jump-to-quote / поиск — прыгают на цитируемое.
- 🚫 Скрытие пунктов настроек (Сферум, Цифровой ID, MAX для бизнеса, Пригласить друзей) — с учётом Turkish-локали.
- 📁 «Только чаты» (`hideChannels`) — в главном списке и в папках.
- 🚪 `gateOpcode5` — блокирует принудительный logout от сервера.
- 📱 SMS-логин — основной и клон.

## ⚠ Известное ограничение AG V1

**Баннер «госуслуги» вверху экрана Настроек — НЕ скрывается.** В GP этот баннер убирался через `hideDigitalId`, но в AG он живёт как view-элемент бинарного layout XML с default-drawable анимацией — точечно перехватить его через smali пока не удалось. Пункт «Цифровой ID» в списке настроек скрывается корректно. Функциональность Госуслуг/ЕСИА не блокируется. Приоритет для AG V1.1 hotfix.

## Установка

1. Скачать нужный APK (arm64 или arm7 — определить через Настройки → О телефоне → Архитектура).
2. Разрешить установку из этого источника.
3. Установить `install -r` (или через файловый менеджер).
4. Для клона — после установки: Настройки → Приложения → MAX Clone → «Очистить данные» → войти по SMS.

## Обновление с GP-мода

Можно ставить AG поверх GP через `install -r` (подпись одна). Но если тумблеры сбросятся — переставить чисто (удалить GP → поставить AG). Данные аккаунта не теряются.
