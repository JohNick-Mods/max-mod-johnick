## 🆕 Новое в 26.23.0 V1

- **🌑 AMOLED-тема (тумблер `amoledTheme`).** Чисто чёрный фон `#000000` для окна, status-bar и navigation-bar — экономит батарею на OLED. Включается **автоматически только когда система в тёмной теме**; в светлой теме не трогает ничего. Категория «Звонки и система», по умолчанию OFF. Известное ограничение: список чатов и всплывающие шторки могут остаться тёмно-серыми (рисуют собственный фон), покрытие ~95%.
- **⌚ Watch-mode: авто-подгонка под Huawei Watch 3/4 Pro и подобные.** При старте приложение измеряет ширину экрана: если меньше 500 px (1.43″, 466×466 @ 326dpi) — принудительно ставится `density=240dpi` (hdpi), layout больше не вылезает за края. Тумблера нет, телефоны не затрагивает. Актуально для сборки **arm7** (часы обычно 32-битные).
- **🎛️ Дев-меню — long-press по версии.** Long-press по строке версии (Настройки → О приложении) открывает нативное меню разработчика MAX. Dispatcher переехал в стоке (`zh2` → `ri2`), Test-crash переякорен (`i81` → `a91`).
- **🔎 Проверка обновлений — short-tap по версии.** Короткий тап по строке версии сверяет текущую версию с GitHub Release и предлагает скачать новую сборку. Dispatcher переехал (`xc` → `gd`). Только по явному тапу — фоновой автопроверки при старте нет.
- **🔓 Антиприсутствие отвязано от «Невидимки».** Тумблер `antiPresence` стал независимым — можно скрывать статус «в сети» / last-seen, не включая «Невидимку».

## 🐞 Что исправлено в 26.23.0 V1

- **🔴 Краш античиталки при скролле/открытии чата.** Критический фикс `NoSuchFieldError`: на новом стоке уехал field скролл-обработчика, из-за чего чат не открывался вовсе. Переякорено.
- **🙈 `hideInviteFriends` / `hideBusinessPromo` снова прячут пункты.** Res-id сменились в стоке (`0x7f110a2a`→`0x7f110a41` и `0x7f110a0c`→`0x7f110a12`), тумблеры обновлены.
- **📬 Антиудалённые сообщения больше не висят непрочитанными (#ANTIDEL-1).** Фикс `iput isRead=1` — восстановленные удалённые сообщения не задирают счётчик непрочитанных.
- **🔧 Полный порт под 26.23.0 (R8-rebump).** Сток бампнут VK (build 6773→6777), 53 apply-скрипта перерезолвены под новую R8-обфускацию.

## ⚠ Известные ограничения

- **AMOLED** — покрытие ~95%: список чатов и всплывающие шторки могут остаться тёмно-серыми.
- **Watch-mode** — элементы визуально мельче на 1.43″ (неизбежно при принудительном hdpi).
- **#FILE-1** — PDF/видео/m4a из основного аккаунта временами отправляются с проблемами. Диагностика продолжается.

## ✅ Сохранено из 26.22.2

- Каналы = сток при любых настройках античиталки (read-mark для каналов всегда уходит на сервер).
- Тумблер `hidePromoWidgets` — скрытие серверных промо-баннеров в списке чатов.
- Анти-удаление переживает жёсткий kill на OEM (Xiaomi/MIUI, Realme, Huawei), метка «· удалено HH:MM».
- «Только чаты» (`hideChannels`) в кастомных папках, `startOnNew` не перетирается.
- `gateOpcode5` — блокировка принудительного logout от сервера.
- Стабильные push на клон между сессиями.
- Баннер «Госуслуги» и пункт «Цифровой ID» скрываются одним тумблером `hideDigitalId`.
- Отправка любых файлов с маскировкой `.bin`, Невидимка-каскад, реакции, скрытие «Пригласить/Найти» в футере, дев-меню MAX.

## 📱 Установка

- Ставится поверх прошлой версии мода: `adb install -r <apk>` — данные сохраняются.
- Первая установка — разрешить неизвестные источники, войти по номеру.
- Клон — после установки один раз: Настройки → Приложения → MAX Clone → Очистить данные → войти по SMS.
- **Минимальный Android — 8.0** (API 26). На 5.0/6.0/7.x не установится — ограничение стока.
