# Цифровой ID (Госуслуги) в MAX — как включить

> **Сборка 26.21.0 V1:** внутренний фикс восстановления ссылок встроен в APK и подтверждён — Цифровой ID создаётся. Если после Госуслуг всё равно открывается браузер с «Не нашли чат» — выполните разовый шаг ниже (разрешить ссылки `max.ru`).

## Проблема
При создании Цифрового ID после авторизации в Госуслугах привязка может не завершаться: вместо возврата в MAX открывается браузер с ошибкой **«Не нашли чат по этой ссылке»**.

## Почему так
Две причины, обе устранены в сборке 26.21.0 V1:
1. **Внутренняя (исправлена в самом APK, действий не требует).** Приватностное отключение трекера в моде побочно ломало штатную обработку входящих ссылок — ответ Госуслуг не доходил до нужного экрана. В этой сборке восстановлено.
2. **Системная — открытие ссылок `max.ru` (требует разового действия).** MAX-мод переподписан другим ключом (без этого мод невозможно установить), поэтому Android **не доверяет** автоматическому открытию ссылок `max.ru` в приложении, и ответ Госуслуг (`https://max.ru/...`) уходит в браузер. Это общее следствие переподписи — касается любого модифицированного MAX. Лечится разовым разрешением ссылок (ниже).

## Решение — разрешить ссылки max.ru (один раз)

### Способ 1 — подсказка в приложении (новое в V2)
При запуске MAX сам один раз покажет окно **«Цифровой ID»** с кнопкой **«Открыть настройки»**. Нажмите её → включите **«Открывать поддерживаемые ссылки»**.

### Способ 2 — вручную
Настройки телефона → **Приложения** → **MAX** → **«Открывать по умолчанию»** → **«Открывать поддерживаемые ссылки»** → включить. Убедитесь, что в списке ссылок есть **max.ru**.

### Способ 3 — через ADB (для продвинутых)
```
adb shell pm set-app-links-user-selection --user 0 --package ru.oneme.app true max.ru
```
Для **клона** пакет другой — `ru.oneme.ap2`:
```
adb shell pm set-app-links-user-selection --user 0 --package ru.oneme.ap2 true max.ru
```

## После этого
Повторите вход через Госуслуги — «Цифровой ID» откроется уже **в MAX**, и привязка завершится.

Настройка привязана к приложению и **сохраняется при обновлениях мода** (`adb install -r`) — повторно включать не нужно. Сбрасывается только при полном удалении приложения или очистке данных.

---

### Проверка статуса (опционально)
```
adb shell pm get-app-links ru.oneme.app
```
- `Selection state: Enabled: max.ru` — ссылки разрешены, Цифровой ID будет работать.
- `Disabled: max.ru` или `max.ru: 1024` без раздела «Enabled» — ссылки ещё не разрешены, выполните один из способов выше.

> Проверено на Samsung S25 (Android 15): сборка 26.21.0 V1 — Цифровой ID создаётся (внутренний фикс встроен). При необходимости один раз разрешите ссылки `max.ru` (шаг выше).
