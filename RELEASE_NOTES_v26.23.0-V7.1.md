# MAX 26.23.0 GP V7.1 — Mods by JohNick

**Дата**: 2026-07-20 · **Сток**: 26.23.0 (build 6777) · **Мин. Android**: 8.0 (API 26)

Обновление поверх V7: правка ввода SMS-кода на смарт-часах, AMOLED-тема теперь включается сама при системной тёмной теме, и починен push у клона. Ставится через `install -r` без потери данных.

## 🐞 Изменения V7.1

- **⌚ SMS-код на часах — вводится последняя цифра.** На смарт-часах (узкий экран) при вводе кода подтверждения последняя ячейка уезжала за край — фокус перепрыгивал назад, и код нельзя было дописать. Теперь ширина ячеек кода считается с учётом watch-плотности: все 6 ячеек умещаются, последняя цифра принимается. На телефонах и планшетах поведение не изменилось (байт-в-байт сток).
- **🌑 AMOLED-тема — авто при тёмной теме.** Раньше чёрный фон нужно было включать тумблером вручную. Теперь при системной тёмной теме AMOLED применяется сам (тумблер остаётся — можно вернуть штатную тёмно-серую). Покрытие ~95% (окно / status-bar / nav-bar; список чатов и шторки могут остаться тёмно-серыми).
- **📲 Push у клона — стабилен.** У второго аккаунта (клон) push мог не приходить: идентификатор устройства клона либо совпадал с основным, либо пересоздавался при каждом запуске, и токен уведомлений устаревал. Теперь клон генерирует собственный идентификатор один раз и больше его не меняет — уведомления приходят стабильно. Основного приложения правка не касается.

## Установка

Ставится поверх V7 через `install -r` — данные сохраняются. Клон рядом с оригиналом.

- `MAX_26.23.0_arm64_V7.1.apk` — современные телефоны (2018+). **Если не уверены — этот.**
- `MAX_26.23.0_arm7_V7.1.apk` — старые 32-битные устройства.
- `MAX_26.23.0_arm64_V7.1_clone.apk` / `MAX_26.23.0_arm7_V7.1_clone.apk` — клон для второго аккаунта.
