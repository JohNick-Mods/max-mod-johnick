# MAX 26.23.0 GP V2 — Mods by JohNick

**Дата**: 2026-07-17 · **Сток**: 26.23.0 (build 6777) · **Android**: 8.0+ (API 26+)
**APK**: 4 файла (arm64 + arm7 × main + clone) · Universal (GP) · Подпись v2+v3

Точечный релиз поверх V1: исправление проверки обновлений + новая opt-in автопроверка. Ставится поверх V1 через `install -r` без потери данных.

---

## 🆕 Новое в 26.23.0 V2

- **🔎 Исправлена проверка обновлений.** Раньше короткий тап по версии показывал «доступно обновление», даже когда установлена самая свежая сборка. Причина — устаревшая жёстко-зашитая строка версии для сравнения с mirror-тегом. Теперь значение версии пересобирается из единого источника при каждой сборке: при актуальной версии честно отвечает «Обновлений не обнаружено».
- **🔔 Автопроверка обновлений — новый чекбокс (по умолчанию ВЫКЛючен).** В окне проверки обновлений добавлен пункт «Автоматически проверять при запуске (не чаще раза в сутки)». Пока не включён — приложение при старте **не делает ни одного сетевого запроса** проверки (как в V1). При включении: раз в 24 часа при старте — **тихая** проверка; диалог появляется **только если новая версия действительно есть**. Без фоновых планировщиков (`AlarmManager`/`WorkManager`/`JobScheduler`) — только однократная inline-проверка при запуске под гейтом «включено + прошли сутки».

## ⚠ Изменение в 26.23.0 V2

- **⌚ Watch-mode временно отключён.** Режим авто-адаптации под смарт-часы, заявленный в V1, по факту не собирался в APK ещё в V1 (на телефонах не активируется — дефект был незаметен). Убран, чтобы не тащить неработающий код. Полноценная реализация — в следующей версии.

## ✅ Сохранено из V1 (без изменений)

- 🌑 **AMOLED-тема** (`amoledTheme`) — чёрный `#000000` фон в тёмной теме, авто только при системной тёмной. Покрытие ~95%.
- 🔓 **Антиприсутствие** (`antiPresence`) — независимый тумблер.
- 🔴 Фикс краша античиталки при открытии чата · 🙈 `hideInviteFriends`/`hideBusinessPromo` · 📬 антиудалённые не висят непрочитанными (#ANTIDEL-1).
- 🎛️ Дев-меню (long-press) · 🔎 проверка обновлений (short-tap) · 📱 SMS-регистрация (User-Agent 26.23.0).
- 🛡️ Анти-удаление (4 слоя + офлайн-персист), «Только чаты», gateOpcode5, стабильный push на клон, hideDigitalId.

## 📱 Установка

- Ставится поверх прошлой версии мода (включая V1): `adb install -r <apk>` — данные сохраняются.
- Первая установка — разрешить неизвестные источники, войти по номеру.
- Клон — после установки один раз: Настройки → Приложения → MAX Clone → Очистить данные → войти по SMS.
- **Минимальный Android — 8.0** (API 26).
