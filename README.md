## Актуальная версия — **26.21.1 V1** (build 6761)

Релиз поверх 26.21.0. Ключевое: **SMS-код при регистрации теперь приходит на новую установку и на клон** (в 26.21.0 сервер MAX не отправлял SMS переподписанным сборкам); плюс hotfix скрытия чатов «Избранное» / «Коды подтверждения» / «MAX» в самом списке.

- 📱 **SMS-fix при регистрации.** С версии MAX 26.20+ сервер гейтит SMS-код по версии клиента в UserAgent — переподписанный мод / клон при первой регистрации не получал код. В 26.21.1 введён условный спуфинг: до логина клиент представляется как легаси-версия `26.17.1 (6712)` (SMS проходит), после первого `logged_in` персистентно (`SharedPreferences "app_flags"`, ключ `verReal`) переключается на реальную `26.21.1 (6763)`. Уже залогиненных пользователей не затрагивает (латч мгновенный на первом WS-коннекте). Тумблера нет — работает всегда автоматически.
- 🔧 **Hotfix folder-hide.** Фильтр перенесён на общий узел AsyncListDiffer — истинный chokepoint любых async submit-путей, покрывает все чат-адаптеры. Модель UI recent-chats (`Lf03`) добавлена в фильтр (в 26.21.0 обрабатывалась только эмит-модель Ly66). Официальный чат «MAX» распознаётся во всех восьми омоглиф-комбинациях Latin/Cyrillic M/А/Х. Проверено на Samsung S25.
- ✅ Все функции 26.21.0 сохранены — 15 тумблеров, скрытый жест long-press по номеру версии, always-on приватность, 13/13 нейтрализованных угроз.
- 📌 Известное ограничение: в режиме Невидимка ручная пометка чата непрочитанным откатывается серверным sync-путём. Требуется отдельный per-chat реестр с TTL — вынесено на следующий релиз.

## Установка

**Обновление с прошлой версии мода:** ставится поверх (подпись та же — cert SHA-1 `d6a7d757…1298b`). **Не удаляйте** предыдущую версию — `install -r` сохраняет данные.

**Первая установка:**
1. Разрешить установку из неизвестных источников (Настройки → Безопасность).
2. Скачать нужный APK из [Releases](../../releases/latest).
3. Открыть APK на телефоне → «Установить».
4. Войти в свой аккаунт (по номеру телефона как обычно).

**Если стоит оригинальный MAX:** удалите перед установкой основного мода (пакет совпадает — `ru.oneme.app`). Клон можно ставить рядом с оригиналом — пакеты разные.

## 15 переключателей в настройках

Долгий тап по заголовку **«О приложении»** (Настройки → О приложении) откроет диалог с тумблерами. Все по умолчанию **OFF** — до первого включения мод ведёт себя идентично стоку (антидетект).

### 🔒 Приватность
- **antiDelete** — антиудалялка (4 слоя + маркер «· удалено»).
- **invisible** — «Невидимка»: совмещённый тумблер — typing / read / presence не передаются, плюс режим «Призрак» (сервер видит вас offline).

### 📁 Скрытие
- **hideAll** — скрыть папку «Всё».
- **hideSferum** — скрыть вкладку «Сферум» в полоске **и** пункт «Войти в Cферум» в настройках.
- **hideFavorites** — 🆕 скрыть папку «Избранное».
- **hideBusinessPromo** — скрыть «MAX для бизнеса» в Settings.
- **hideInviteFriends** — скрыть «Пригласить друзей» в меню.

### 🧭 Навигация
- **reorderTabs** — «Чаты» первыми вместо «Контактов».
- **startOnNew** — старт на папке «Новые» (основной + клон).

### 🔌 Системное и звонки
- **autoStart** — управление автозапуском в фоне (default OFF).
- **improveNotifications** — восстанавливает WS при смене сети (Wi-Fi ↔ мобильная).
- **backCameraForNote** — видео-кружки с основной камеры вместо фронтальной.
- **debugMenu** — разблокирует скрытое **инженерное меню звонков** (кнопка «Ещё» внутри звонка).

### 📵 Сеть / оператор / профиль
- **blockMobileIdVerify** — скрыть номер от оператора (Header Enrichment / Mobile ID). На «Цифровой ID» **не влияет**.
- **hidePhoneNumber** — скрыть номер в профиле (только отображение).

## Скрытый жест — dev-меню MAX

**Long-press по номеру версии** (Настройки → строка с версией → долгий тап) открывает **нативный экран dev-настроек MAX** (`:settings/dev`). В стоке фича есть, но загейчена под dev-канал сборки — мод снимает гейт. Отличается от тумблера «Debug-меню звонков»:

| Что открывает | Как включается | Где показывается |
|---|---|---|
| **Нативное dev-меню MAX** | Long-press по номеру версии | Настройки → строка версии |
| **Debug-меню звонков** (тумблер `debugMenu`) | Включить тумблер | Внутри звонка → кнопка «Ещё» |

Short-tap по номеру версии не трогается — работает как в стоке.

## Always-on приватность и удобства (без тумблера)

Эти правки включены безусловно — либо приватность по умолчанию, либо системные удобства.

- **Скриншоты и запись экрана разрешены везде.** Убран `FLAG_SECURE (0x2000)` в двух местах (`MainActivity`, `v29`) — приложение больше не блокирует захват экрана в защищённых окнах.
- **IP-адрес не отправляется в телеметрию соединения.** В `c2/k10.smali` вырезана строка `put("ip", …)`; `connection_type` и `vpn` сохраняются (безобидны).
- **Автоповорот экрана следует настройке устройства.** `MainActivity.V()` больше не форсирует portrait — `SCREEN_ORIENTATION_USER`. `VideoWebViewScreen` работает независимо (видео вёрстанное фиксированно).
- **Диалог «Разрешить ссылки max.ru» (once-guard).** При запуске один раз показывает подсказку с переходом в «Открывать по умолчанию → Открывать поддерживаемые ссылки» — чинит переподписной трюк с App Links для Цифрового ID.

## Антиудалялка — 4 слоя

1. **L3 — Central WS chat-event**. Live-канал, когда собеседник удаляет прямо сейчас и приложение онлайн.
2. **L4 — Room ORM body-clear**. Локальный SQL `UPDATE text=NULL, attaches=NULL, status=10` — блокируется на входе.
3. **V3 persistence (read-layer un-filter)** — SQL-бинды чтения. При `antiDelete=ON` удалённые сообщения остаются видимыми после ре-синка.
4. **Visible marker** — у удалённых сообщений в строке времени дописывается «· удалено».

Общий гейт `antiDelete`: **ON** → защита активна; **OFF** → точное поведение стока. Реакции-эмодзи у соседних сообщений рендерятся корректно.

## Цифровой ID (Госуслуги) — если не открывается

Внутренний фикс из 26.20.2 встроен в APK — обычно Цифровой ID создаётся сразу. Но из-за переподписи мода Android может не доверять автоматическому открытию ссылок `max.ru` — ответ Госуслуг уйдёт в браузер с «Не нашли чат». Лечится один раз:

**Способ 1 — подсказка в приложении.** При запуске MAX сам один раз покажет окно «Цифровой ID» с кнопкой **«Открыть настройки»** → включить «Открывать поддерживаемые ссылки».

**Способ 2 — вручную.** Настройки телефона → **Приложения → MAX → «Открывать по умолчанию» → «Открывать поддерживаемые ссылки»** → включить (`max.ru` должен быть в списке).

**Способ 3 — ADB:**
```bash
# основной
adb shell pm set-app-links-user-selection --user 0 --package ru.oneme.app true max.ru
# клон
adb shell pm set-app-links-user-selection --user 0 --package ru.oneme.ap2 true max.ru
```

Настройка сохраняется при обновлениях мода (`install -r`).

- **Контакты в фоне** — выгрузка адресной книги при foreground (`x10.b()`) заглушена.
- **Загрузка при boot** — гейтится тумблером `autoStart` (OFF по умолчанию).

## Уведомления — если не приходят

MAX получает push через **системные сервисы** (FCM / HMS). Своего канала доставки нет.

1. **Нет GMS/microG → push не придёт никогда**, независимо от мода. Поставьте Google Play Services или microG.
2. **Зарубежный трафик режут операторы** → FCM не пробивается. VPN или Wi-Fi без шейпинга.
3. **Производитель усыпляет приложение** (Xiaomi/Huawei/Samsung/Oppo/Vivo):
   - Настройки → Приложения → MAX → **Батарея → Без ограничений**
   - Xiaomi: **Автозапуск** + **Контроль активности → Нет ограничений**
   - Huawei: **Запуск приложений** → выключить «Автоматическое управление»
   - Samsung: Уход за устройством → Батарея → **Ограничения фона → MAX убрать**
4. Включите тумблер **`improveNotifications`** — восстанавливает WS при смене сети.
