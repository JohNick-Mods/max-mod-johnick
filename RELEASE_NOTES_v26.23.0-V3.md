# MAX 26.23.0 GP V3 — Mods by JohNick

**Дата**: 2026-07-17 · **Сток**: 26.23.0 (build 6777) · **Мин. Android**: 8.0 (API 26)

Точечный релиз поверх V2. Включён **рабочий Watch-mode** — авто-адаптация под смарт-часы. Всё из V2 сохранено без изменений.

## 🆕 Новое в V3

- **⌚ Watch-mode — авто-адаптация под смарт-часы.** При старте мод смотрит на ширину экрана в пикселях. Если она **< 500 px** (Huawei Watch, WearOS и т.п.) — плотность экрана переопределяется на **240 dpi** (hdpi), и интерфейс, рассчитанный на телефон (~360 dp), ужимается до ~310 dp — умещается на маленьком/круглом экране. На телефонах и планшетах (1080 px+) режим **не активируется**. Локализация при переключении плотности **сохраняется**. Без тумблера — полностью автоматически по разрешению экрана.

## ✅ Сохранено из V2

- `MAX_26.23.0_arm64_V3_clone.apk` / `MAX_26.23.0_arm7_V3_clone.apk` — клон для второго аккаунта.
